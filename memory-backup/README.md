Memory Backup

This folder stores manual backups of Copilot memory files so they can be restored if `/memories/` is cleared.

Contents:
- `preferences.md`: user preferences
- `repo/portfolio-self-hosted-runner-checkpoint.md`: repository-specific checkpoint notes

Restore:
1. Recreate the corresponding memory files in `/memories/`.
2. Copy the contents from these backup files into the memory system.
3. Confirm the restored files are visible in `/memories/`.

Recommended habit:
- Update this folder whenever you add or change important memory entries.
